Для добавления нового метода в веб-сервисы:
1. Проверить процедуру / запрос в MS SQL Server
2. Создать в папке Models класс модели данных. Можно создать класс в отдельном файле .cs, либо можно добавить класс в существующий файл CustomerInfo.cs Названия свойств в создаваемом классе должны совпадать с названиями полей выходных данных запроса (или процедуры) в БД.
3. В файле DataAccess.cs (в папке Classes) создать метод, вызывающий нужную процедуру и передающий ей необходимые входные параметры. Тип выходных данных должен соответствовать выходным данным процедуры (запроса), то есть должен быть использован созданный класс модели данных.
4. В файле InfoController.cs (в папке Controllers) создать метод, вызывающий нужный метод класса DataAccess и передающий ей необходимые входные параметры. Также нужно прописать [Route("")]. См. метод GetCustomerByPhone().
5. Rebuild и Run.
6. Сделать Publish и скопировать папку publish-а на сервер IIS. Возможно, перед этим стоит сделать backup работающего сервиса.
