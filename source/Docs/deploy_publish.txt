Добавление InfoCenter
1. В папке c:\inetpub\wwwroot\ создать папку InfoCenter
2. Запустить inetmgr
3. Щелкнуть правой кнопкой мыши по "сайты"
4. Из меню выбрать "Добавить веб-сайт"
5. В окне "Добавление веб-сайта" в поле "Имя сайта" прописать "InfoCenter"
6. В поле "Физический путь" прописать "C:\inetpub\wwwroot\InfoCenter"
7. Проверка. В браузере открыть http://localhost/

Добавление api
1. В папке c:\inetpub\wwwroot\ создать папку api
2. Запустить inetmgr
3. Щелкнуть правой кнопкой мыши по "Default Web-site"
4. Из меню выбрать "Добавить приложение"
5. В окне "Добавление приложения" в поле Псевдоним прописать "api"
6. В поле "Физический путь" прописать "C:\inetpub\wwwroot\web_api_mssql"
7. Проверка. В браузере открыть http://localhost/api/customerbyphone/9900
